"Snowbanks" (demo.app.1) 
======================== 

Программа, демонстрирующая работу с некоторыми классами MFC. 

Включает, среди прочего, работу со следующими элементами:

 - работу с файлами 
 - работу с классом коллекции CStringList 
 - работу с функцией CString::Tokenize() 
 - вывод графики (работу с классом CDC) 
 - работу с классом CRgn 
 - работу с таймером 
 - использование макросов ASSERT и TRACE для вывода отладочной информации 
 - решение проблемы неправильно определяемой при записи в unicode-CString кодировки для кириллического текста 
 - работу со строкой состояния
 
Программа читает текст из тесктового файла и выводит его по одному слову, так что каждое слово медленно "падает" из верхней части окна, отталкиваясь от краев. Слова скапливаются внизу окна, останавливаясь либо у нижней границы окна, либо при соприкосновении с другим словом, "падая" на него сверху. Таким образом, у нижней границы окна создаются "сугробы" из слов. Размер окна программы можно менять, однако при этом содержимое очищается. Это связано с тем, что мне хотелось в этой программе, в отличие от различных типичных примеров из самоучителей по MFC, использующих класс CDocument для сохранения содержимого сосотояний отображения, попробовать ради интереса использовать для этой цели класс, производный от CView, не применяя обработчик OnPaint сообщения WM_PAINT. Таким образом, программу удалось реализовать без использования CDocument, так что данные отображения остаются динамическими и удаляются при автоматически вызываемой при изменении размеров окна перерисовке. Отображаемый текст должен находиться в файле "file.txt" в каталоге программы, по умочанию там находится текст файла readme. Во время работы программы в строке состояния выводится номер текущего слова и общее количество слов в файле.
